## Brafé - Grid

Essa é a segunda parte de um projeto anterior que foi desenvolvido como parte de um estudo de caso para demonstrar técnicas de layout responsivo e utilização de um sistema de grid manual.

### Visão Geral do Projeto

O Brafé - Grid é uma página web que apresenta informações sobre diversos tipos de café brasileiro, incluindo detalhes sobre sua origem, características e locais de venda. O projeto é focado na aplicação de um sistema de grid manual para a criação de layouts flexíveis e responsivos, sem o uso de estruturas mais automatizadas como `grid-template-columns`.

### Implementação do Grid

Ao contrário das abordagens mais convencionais que utilizam `grid-template-columns`, o sistema de grid deste projeto foi implementado manualmente. As classes CSS foram definidas para cada tamanho de coluna desejado, permitindo uma distribuição flexível e adaptável dos elementos na página.

Através do uso de porcentagens e cálculos simples, cada coluna é ajustada dinamicamente para se adaptar ao tamanho da tela do dispositivo, garantindo uma experiência consistente em diferentes dispositivos e resoluções de tela.

### Estrutura do Repositório

- **HTML:** O arquivo HTML contém a estrutura básica da página web, incluindo a definição dos elementos e a organização dos conteúdos.
- **CSS:** O diretório CSS contém dois arquivos CSS:
  - `style.css`: Este arquivo contém estilos gerais aplicados à página, como fontes, cores e margens.
  - `grid.css`: Este arquivo define as classes CSS que implementam o sistema de grid manualmente, permitindo o posicionamento e distribuição dos elementos na página.

### Tecnologias Utilizadas

O projeto foi desenvolvido utilizando HTML e CSS puros, sem a necessidade de bibliotecas ou frameworks externos. Isso permite uma compreensão mais clara e direta do código e das técnicas empregadas, além de garantir uma maior flexibilidade e controle sobre o layout da página.
